# ras-negotiator
[IT] Simulatore di apprendimento attivo basato sul Sistema Reticolare Attivante (RAS). Obiettivo: Migliorare le performance negoziali e il focus operativo. Funzionalità: Trigger cognitivo per audit mentale, audio separato dalla navigazione per calibrazione di precisione, navigazione manuale.

[EN] Active learning simulator based on the Reticular Activating System (RAS). Objective: Improve negotiation performance and operational focus. Features: Cognitive triggers for mental audit, audio separated from navigation for precision calibration, manual control for learning rhythm.
